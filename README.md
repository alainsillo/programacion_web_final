## Objetivo del proyecto final:

Realizar 5 paginas web: 

1.-Biografia: Hacer una biografia donde coloquen foto de ustedes, sus pasatiempos, metas a corto mediano y largo plazo y donde mencionen porque estan estudiando la carrera de Ingeniería en Sistemas.

2.- Glosario: Colocar definiciones o para que se utiliza lo visto en el curso de CODEDEX. 

3.- CRUD. Hacer la parte del html y css y si es posible implementar el JS para su funcionalidad (Aunque es opcional).

4.- Pagina de Banda Musical. Hacer una pagina de tu banda favorita.

5.- Pagina de Herramientas. Hacer una pagina con listas o tablas, donde enlistes las herramientas o recursos a utilizar en HTML y CSS. Pueden ser desde plataformas, paginas web, tutoriales hasta videos de youtube. Cada recurso compartido deberá dirigirte a travez de un enlace al sitio correspondiente.

POR ULTIMO. Generar una pagina que actue como principal (index.html) y que tenga links o botones que te lleven a paginas creadas. Las paginas ya creadas tambien tendrán un link que te regrese a la principal.

Los estilos y formato de las paginas deberán ser el mismo, por lo cual se necesitara un solo archivo styles.css.
