Glosario de términos HTML y CSS

Selector
En CSS, un selector se usa para elegir qué parte del HTML será afectada por los estilos. Por ejemplo, el selector p cambia todos los párrafos, mientras que #titulo cambia un elemento con un ID específico.
Atributo
Los atributos dan información adicional a una etiqueta HTML. Por ejemplo, <img src="imagen.jpg" alt="foto"> tiene los atributos src (la ruta de la imagen) y alt (el texto alternativo).
ID
El atributo id identifica un elemento único en una página. Sirve para crear enlaces internos o aplicar estilos específicos, por ejemplo: <h1 id="titulo">Mi título</h1>.
Clase (class)
Una class agrupa varios elementos para darles el mismo estilo. Por ejemplo, <p class="importante">...</p> permite aplicar un mismo formato a varios párrafos.
Etiqueta
Una etiqueta es la base del HTML. Define el tipo de contenido, como un párrafo (<p>), una imagen (<img>), o un enlace (<a>).
Padding
En CSS, el padding es el espacio interno entre el contenido y el borde de un elemento. Por ejemplo, si un cuadro tiene texto, el padding separa el texto del borde del cuadro. Se controla con propiedades como padding, padding-top, etc.
Alt
El atributo alt describe el contenido de una imagen. Es útil cuando la imagen no carga o para accesibilidad; por ejemplo: <img src="foto.jpg" alt="Retrato de persona">.
Href
El atributo href se usa en los enlaces (<a>) para indicar la dirección a la que se va a dirigir el usuario: <a href="https://ejemplo.com">Ir a ejemplo</a>.
